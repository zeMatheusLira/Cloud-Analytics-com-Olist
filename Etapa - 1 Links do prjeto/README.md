# Cloud Analytics com Olist – Business Intelligence

Este repositório apresenta o projeto **Cloud Analytics com Olist**, desenvolvido como atividade prática de Engenharia de Dados e Business Intelligence. O projeto contempla todas as etapas de construção de um ambiente analítico, desde a modelagem dimensional dos dados até a disponibilização de dashboards interativos para apoio à tomada de decisão.

O conjunto de dados utilizado é o **Olist Brazilian E-Commerce Dataset**, amplamente empregado em estudos de análise de dados e Data Warehousing.

---

# Dashboard Interativo

O resultado final das análises pode ser acessado por meio do dashboard desenvolvido no **Apache Superset (Preset Cloud)**.

🔗 **Acesse o Dashboard:**

https://98d7e14b.us1a.app.preset.io/superset/dashboard/8/?native_filters_key=zfPbleROXcE

O dashboard reúne indicadores estratégicos relacionados à operação da Olist, permitindo análises interativas sobre:

- Receita por categoria de produtos;
- Volume de pedidos por estado;
- Ticket médio por estado;
- Atrasos logísticos entre estados de vendedores e clientes;
- Satisfação média dos clientes;
- Distribuição dos meios de pagamento.

Além disso, é possível utilizar filtros para explorar diferentes categorias de produtos, tornando a análise mais dinâmica.

# Objetivo do Projeto

O principal objetivo deste trabalho foi transformar dados transacionais do marketplace Olist em informações estratégicas por meio da construção de um Data Warehouse e de dashboards analíticos.

Com isso, tornou-se possível identificar padrões de vendas, gargalos logísticos, comportamento dos consumidores e oportunidades de melhoria na operação, demonstrando a aplicação prática de conceitos de Engenharia de Dados, ETL, Modelagem Dimensional e Business Intelligence.

# Principais Análises

O dashboard permite responder perguntas de negócio como:

- Quais categorias geram maior receita?
- Quais estados concentram maior volume de pedidos?
- Onde estão os principais gargalos logísticos?
- Como a logística impacta a satisfação dos clientes?
- Qual o comportamento dos meios de pagamento utilizados na plataforma?

Essas análises fornecem informações relevantes para apoiar decisões estratégicas relacionadas à logística, vendas e experiência do cliente.
